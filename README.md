# Spectre

A spectral path tracer that focuses on physically accurate light behavior. As of now, it features:

- **Continuous wavelength sampling:** instead of discretizing spectral distributions, this renderer accepts continous representations in the form of functions (sigmoids, RBFs etc.), being possible via single-wavelength rays instead of array SPDs; hero wavelength rays were implemented but the performance hit did not justify the small reduction in chromatic noise.
- **Precise rendering of curved surfaces:** no need to triangulate, this implementation focuses on pure geometric definitions; of course, that limits the types of shapes it is able to render, but more may be added eventually.
- **Spectral phenomena:** as can be seen in the image, chromatic aberration generated by light diffraction through glass is accurately portrayed; thin film interference may be added as well.
